# Vaderfiguur landingspagina

Statische React-site (Vite + Tailwind). Losgekoppeld van Base44.

## Lokaal draaien
```
npm install
npm run dev
```

## Bouwen
```
npm run build
```
Output staat in `dist/`.

## Hosten op Netlify
Build command: `npm run build`
Publish directory: `dist`
Deze staan al in `netlify.toml`, dus Netlify pakt ze automatisch op.
